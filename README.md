# 🧠 Engenharia de Prompts

Projeto escolar interativo desenvolvido com o objetivo de ensinar os conceitos básicos de **engenharia de prompts** de forma prática, visual e didática.

---

## 📌 Sobre o Projeto

Este projeto é uma página web educativa que apresenta:

* Conceitos fundamentais sobre Inteligência Artificial
* Estrutura de prompts eficazes
* Métodos e boas práticas
* Exemplos comparativos (prompt ruim vs bom)
* Quiz interativo para fixação
* Gerador de prompts inteligente

A proposta é ajudar alunos a entender como se comunicar melhor com IAs como o ChatGPT.

---

## 🎯 Objetivos de Aprendizagem

* Compreender o que é um prompt
* Aprender a estruturar comandos claros e eficientes
* Desenvolver pensamento crítico ao usar IA
* Praticar a melhoria contínua de prompts (iteração)
* Entender o impacto do contexto e do formato nas respostas

---

### 🧪 Quiz Interativo

* 6 questões sobre engenharia de prompts
* Correção automática
* Feedback explicativo para cada resposta

---

### ⚡ Gerador de Prompt Inteligente

Permite criar prompts completos com base em:

* Papel da IA
* Tarefa
* Contexto
* Formato
* Limitações

🔧 Exemplo de saída:

```
Aja como um professor de história. Sua tarefa é explicar a Segunda Guerra Mundial. 
Considere o seguinte contexto: para um aluno do 9º ano. 
Responda no formato: em tópicos. 
Siga estas restrições: linguagem simples e até 10 linhas. 
Seja claro, objetivo e didático.
```
